# Book Library
Aplikasi ini dibuat untuk mengikuti lomba Internal Android IDN Boarding School

## Tentang Aplikasi
Aplikasi ini dibuat menggunakan beberapa library diantaranya :

- Firebase Authentication
- Login Session
- GSON
- Glide
- Recycler View
- Nav Graph
- Navigation UI
